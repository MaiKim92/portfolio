Rushify Card Maker

Rushify is a custom card generator for the popular card game Yu-Gi-Oh! Rush Duel. The user inputs information to be displayed on the card, upon which the website will generate the card image, which the player can then save to their computer. The user can also upload an Excel worksheet with card information to have the website batch generate multiple cards at once.
